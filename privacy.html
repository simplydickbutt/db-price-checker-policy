Privacy Policy

Last updated: July 17, 2025

DB Price Checker

DB Price Checker is a Chrome Extension that converts fiat currency prices displayed on websites into the real-time value of the $DICKBUTT memecoin. This is done entirely on the client-side for user convenience and entertainment.

Data Collection

We do not collect, store, or transmit any personal data, browsing history, or user-identifiable information.

  • No user data is collected.
  • No tracking or analytics scripts are included.
  • No external servers are used to store or process user information.

Permissions

The extension requests the following permissions only to function as intended:

  • "activeTab" – to read and modify the prices displayed on the webpage.
  • Access to public cryptocurrency APIs for live price conversion.

These permissions are strictly used for converting fiat prices to memecoin value.

Third-Party Services

We may use public APIs (e.g., CoinGecko) to fetch real-time memecoin prices. These services may have their own privacy policies. No user information is sent to them.

Changes to This Policy

This Privacy Policy may be updated in the future. Any changes will be reflected on this page with a revised date.

Contact

If you have any questions or concerns about this Privacy Policy, feel free to contact us via GitHub or the Chrome Web Store support page.
